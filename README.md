# Is Window

Checks if the given value is a window object.

## Install

```bash
# npm
npm install --save-dev @elonehoo/is-window

# yarn
yarn add  --dev @elonehoo/is-window

#pnpm
pnpm install --save-dev @elonehoo/is-window
```
